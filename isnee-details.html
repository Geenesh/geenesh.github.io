Project Overview

ISNEE is a comprehensive event management platform specifically designed for motorsport enthusiasts and racing communities. The system streamlines the entire event lifecycle from initial planning and participant registration to race day coordination and post-event analytics.

The platform addresses the unique challenges of motorsport event organization by providing specialized tools for race registrations, participant management, timing systems integration, safety protocol management, and real-time event coordination, making it an essential tool for racing organizers and participants alike.

Technology Stack

React.js NestJS PostgreSQL TypeScript JWT Authentication Socket.io Redis Docker AWS

Key Features

Event Creation & Management

Comprehensive event setup with track details, race categories, timing schedules, safety requirements, and participant capacity management.

Registration System

Streamlined participant registration with vehicle verification, license validation, insurance checks, and category assignment.

Real-time Race Management

Live race coordination with timing integration, position tracking, incident reporting, and real-time leaderboard updates.

Participant Portal

Dedicated participant dashboard for registration management, event schedules, race results, and communication with organizers.

Safety & Compliance

Integrated safety protocol management, technical inspection tracking, and compliance verification for motorsport regulations.

Analytics & Reporting

Comprehensive event analytics, performance reports, participant statistics, and financial tracking for event organizers.

Development Responsibilities

  • Frontend Development
    Built responsive React.js application with TypeScript for type safety and improved developer experience.
  • Backend API Development
    Developed robust NestJS backend with modular architecture, implementing RESTful APIs and real-time WebSocket connections.
  • Database Architecture
    Designed and implemented PostgreSQL database schema optimized for complex motorsport event data and participant relationships.
  • Real-time Features
    Implemented Socket.io for real-time race updates, live timing data, and instant communication between organizers and participants.
  • Authentication & Authorization
    Built secure authentication system with role-based access control for organizers, participants, and officials.
  • Performance Optimization
    Implemented Redis caching for improved performance and optimized database queries for handling large-scale event data.
  • DevOps & Deployment
    Containerized application with Docker and deployed on AWS with automated CI/CD pipelines for reliable production deployment.
