Project Overview

Namma Kutira is a comprehensive rental property management mobile application designed to streamline interactions between landlords and tenants. The platform facilitates efficient property oversight, rent collection, maintenance request handling, and communication management, making property management more organized and transparent for all stakeholders.

The application addresses common challenges in property management by providing digital solutions for rent tracking, maintenance coordination, tenant communication, and property documentation, ultimately improving the rental experience for both property owners and tenants.

Technology Stack

React Native Node.js Express.js MongoDB JWT Authentication Firebase Push Notifications AWS S3 Redux

Key Features

Property Management

Comprehensive property listing management with photos, descriptions, rental terms, and tenant assignment capabilities.

Rent Collection & Tracking

Automated rent reminders, payment tracking, receipt generation, and overdue payment notifications for seamless financial management.

Maintenance Requests

Digital maintenance request system allowing tenants to report issues with photos and enabling landlords to track repair progress.

Tenant Communication

In-app messaging system for direct communication between landlords and tenants, with notification support for important updates.

Document Management

Secure storage and management of lease agreements, property documents, and tenant information with easy access and sharing.

Analytics & Reporting

Property performance analytics, rent collection reports, and maintenance cost tracking for informed decision making.

Development Responsibilities

  • Mobile App Development
    Built cross-platform mobile application using React Native with intuitive user interfaces for both landlords and tenants.
  • Backend API Development
    Developed RESTful APIs using Node.js and Express.js for user management, property operations, and notification services.
  • Database Design & Implementation
    Designed and implemented MongoDB database schema for efficient storage of property data, user information, and transaction records.
  • Authentication & Security
    Implemented secure user authentication with JWT tokens and role-based access control for landlords and tenants.
  • Cloud Integration
    Integrated AWS S3 for document storage and Firebase for push notifications to enhance user engagement and data management.
  • Payment Integration
    Integrated payment gateway solutions for secure rent collection and automated payment processing.
  • Testing & Deployment
    Conducted comprehensive testing across iOS and Android platforms and managed deployment to app stores.
