Project Overview

Kridas is a global sports networking and career platform that provides expansion, collaboration, sponsorship and marketing opportunities for all stakeholders within the sporting industry. The platform brings together athletes, coaches, clubs, sponsors, and sports enthusiasts to create their unique sporting profiles and build a global network.

The platform serves as a comprehensive ecosystem for the sports industry, facilitating connections, career development, and business opportunities while promoting sports culture and community building across different disciplines and geographical boundaries.

Development Responsibilities

  • Full Stack Development
    Built comprehensive web and mobile applications using React and React Native with focus on cross-platform compatibility and performance.
  • Backend Architecture
    Developed scalable backend systems using Node.js and Express.js with GraphQL APIs for efficient data querying and real-time features.
  • Database Design
    Designed and implemented MongoDB database schema for complex sports networking data, user relationships, and content management.
  • Social Features Implementation
    Built social networking features including user profiles, feeds, messaging, event management, and community interaction tools.
  • Performance Optimization
    Implemented Redis caching, database query optimization, and code splitting to ensure fast loading times and smooth user experience.
  • DevOps & Deployment
    Containerized applications using Docker and managed deployment infrastructure to ensure reliable and scalable platform operation.
  • Feature Development
    Developed specialized sports industry features including athlete portfolios, sponsorship matching, event coordination, and marketplace functionality.
